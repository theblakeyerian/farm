BRASSICA CLUBROOT WARNING: Before establishing A4 and A7, confirm no history of clubroot on the Edgehill property. If present, this pathogen persists for 20 years and makes brassica production in that ground effectively impossible. Test with indicator plants (Chinese cabbage) before committing to these plots. If clean, maintain strict 3-year rotation and keep pH at or above 7.0 in brassica plots to suppress clubroot development.

Market garden infrastructure — tunnels, tarps, row cover, and irrigation

Caterpillar Tunnels

50-foot commercial tunnel kits oriented across the 50-ft width of each plot — one tunnel spans all 32 planted rows simultaneously. Multiple tunnels positioned at intervals along the 100-ft plot depth. This gives you the ability to cover specific row zones (early transplants, overwintering crops, seedling protection) without covering the entire plot.

Silage Tarps

One 50×100 ft tarp per plot. Your primary bed-prep and weed-termination tool. In September, get all 14 tarps down immediately on any area being prepared for perennial planting. 3–6 weeks under a tarp in September heat kills existing vegetation, terminates cover crops, and leaves clean moist soil ready for planting — no herbicide, no tillage beyond your BCS pass.

Automated Drip Irrigation System

One zone per plot (14–16 zones total), controlled by soil moisture sensors, managed through a custom app built on ESP32 + Firebase + React Native. The physical system is permanent infrastructure; the controller nodes and app can be built and added incrementally.

System architecture (well pump → field → app) Well pump → Pressure tank → Master shutoff
  → 150-mesh disc filter → 12 PSI pressure regulator
    → 16-station zone manifold (solenoid valves)
      → Plot header (1/2-in poly mainline)
        → Drip tape runs (32 rows × 50 ft per plot)
        → ESP32 node reads soil moisture sensors → triggers valve relay
        → MQTT → Raspberry Pi (local broker) → Firebase → React Native app

• Schedule fallback — if sensor fails or MQTT drops, run on time schedule until fixed
• Alerts (push) — zone runs >30 min (possible leak), sensor reads 0% (sensor failure), pressure drops (line break)
